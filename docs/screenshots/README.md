# Screenshots

The launcher captures these automatically on real hardware into
`sdmc:/AndroidHorizonNX/screenshots/`:

| File | When it's taken |
|------|-----------------|
| `ui_menu.png` | APK browser, ~3 s after launch (animations settled) |
| `ui_about.png` | About screen, once the GitHub avatar has loaded |

After a test run, copy the PNGs from the SD card into this folder — the main
README embeds them from here.

Gameplay/loading-screen screenshots (`ui_loading.png`, `ui_result.png`,
`game_frame*.png`) now live in the
[AHNX-Translation-Core](https://github.com/AndroidHorizon/AHNX-Translation-Core)
repo instead, since those screens are captured by the engine, not the launcher.
